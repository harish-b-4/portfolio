🚀 Personal Portfolio Website

This is my personal portfolio website built using React + Vite and styled with Tailwind CSS. The portfolio is designed to showcase my skills, projects, and background as a MERN stack learner in a clean, responsive, and modern UI.

📌 Pages Included

• Home – Highlights my role and technical skill set

• About – Displays my personal details and educational background

• Projects – Showcases the projects I completed during my MERN course journey

• Contact – Contains my contact information and a contact form (backend integration planned)

🛠 Tech Stack

• React

• Vite

• Tailwind CSS


✨ Key Features

• Fully responsive design

• Modern UI with Tailwind CSS

• Component-based architecture

• Clean and organized project structure

• Scalable setup with future backend integration planned for the contact form

🔮 Future Enhancements

• Backend integration for the contact form

• Email or database support for form submissions

• Additional animations and performance optimizations
